02

音标学习 · PPP教学法

Presentation → Practice → Production，三步掌握核心音标

P

Presentation

呈现 · 认知

通过生动的故事情境，引出四个核心元音音标，让学生在情境中感知发音规则，建立初步认知。

P

Practice

练习 · 巩固

多样化的互动练习，如音标Bingo、吃包子迷宫等游戏，让学生在玩中学，轻松掌握发音技巧。

P

Production

产出 · 运用

口语输出环节，学生运用所学音标进行实际表达，将知识转化为能力，实现真正的语言输出。

03

语法学习 · TTT教学法

Test → Teach → Test，闭环式语法习得之旅

T

Test

测试 · 诊断

课前诊断测试，了解学生对名词复数规则的现有认知水平，为个性化教学提供依据。

T

Teach

讲授 · 建构

系统讲解可数名词变复数的规则变化，通过故事情境和实例演示，让学生在理解中建构语法知识。

T

Test

测试 · 检验

课后效果检验，评估学生对复数规则的掌握程度，形成学习闭环，确保学习效果。

📚 可数名词变复数 · 核心规则

系统掌握名词复数变化规律，夯实语法基础

➕

一般情况加 -s

  • cat→cats
  • book→books
  • dog→dogs
🎵

清辅音后读 /s/

  • map→maps
  • bike→bikes
  • cup→cups
🎸

浊辅音/元音后读 /z/

  • bed→beds
  • bag→bags
  • day→days
